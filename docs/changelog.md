# Changelog

This changelog tracks user-facing changes to the current live Klippventurer project line.

## v0.2.2 - Upcoming

### Changed

- Clarifies that `main` is the current live v0.2.x manual/soldered host install workflow.
- Links from the README to the preview **USB Install Branch** (`next/firmware-package`) for next-generation stock-like USB installer work.
- Centers the README navigation links under the Discord badge for GitLab rendering.
- Updates the Adventurer 3 installation guide for Raspberry Pi Imager 2.0.2 and Raspberry Pi OS Bookworm paths.
- Updates setup commands to use GitLab-hosted Klippventurer URLs.
- Temporarily keeps KAMP setup out of the manual installation flow pending Discord context and a proper fix for the issue that prompted its removal.
- Updates STM32 flashing instructions and includes an example successful `stm32flash` output.
- Adds clearer post-install calibration guidance.
- Updates the soldered-install recovery firmware link.

### Notes

- This remains the manual/soldered host install project line.
- N32 MCU support and the stock-like USB installer are not included in v0.2.2; those are being developed on the preview USB installer branch.
